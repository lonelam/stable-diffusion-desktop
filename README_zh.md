# Stable Diffusion桌面版

Stable Diffusion桌面版是一个Electron桌面应用，它封装了Stable Diffusion WebUI所需要的所有依赖和配置过程。这让你可以像使用桌面端App那样快速安装并上手Stable Diffusion，而无需学习细节和复杂的参数作用。

## 路线图

- [x] Nvidia 支持
- [ ] AMD 支持
- 目前正在进行中:
  - [ ] AMD + Windows 支持
  - [ ] AMD + Mac Intel 支持
- [ ] Mac Intel 支持
- [ ] Mac M1 支持
- [ ] Windows 支持
